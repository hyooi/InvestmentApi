## 사용법

1. 앱 실행

```
 ./gradlew bootRun
```

2. H2 URL

```
http://localhost:6549/h2
```

## 사용 기술

### Language

- Java11

### Framework / Library

- Spring Boot2
- JPA & Hibernate
- Junit5

### Database

- H2(in-memory)

### Etc

- Jacoco
- Lombok
- Gradle
- Github action
